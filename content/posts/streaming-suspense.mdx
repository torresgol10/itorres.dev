El streaming permite enviar partes de la página al cliente mientras otras se siguen cargando. Esto mejora significativamente el Time to First Byte (TTFB) y la experiencia del usuario.

## Loading UI con Suspense

Usa `loading.tsx` o `<Suspense>` para mostrar estados de carga:

```tsx
import { Suspense } from "react"
import { PostList } from "./PostList"
import { PostListSkeleton } from "./PostListSkeleton"

export default function Page() {
  return (
    <main>
      <h1>Blog</h1>
      <Suspense fallback={<PostListSkeleton />}>
        <PostList />
      </Suspense>
    </main>
  )
}
```

## Streaming Paralelo

Puedes hacer streaming de múltiples componentes en paralelo:

```tsx
export default function Dashboard() {
  return (
    <div className="grid grid-cols-2 gap-4">
      <Suspense fallback={<ChartSkeleton />}>
        <RevenueChart />
      </Suspense>
      <Suspense fallback={<TableSkeleton />}>
        <LatestOrders />
      </Suspense>
    </div>
  )
}
```

## Conclusión

El streaming con Suspense transforma la experiencia de carga de tu aplicación, mostrando contenido progresivamente en lugar de una pantalla en blanco.
