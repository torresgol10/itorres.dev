Server Actions son una de las características más poderosas de Next.js 14. Te permiten ejecutar código del servidor directamente desde tus componentes de React.

## ¿Qué son Server Actions?

Son funciones asíncronas que se ejecutan en el servidor pero pueden ser invocadas desde el cliente. Esto elimina la necesidad de crear API routes para operaciones simples.

```tsx
"use server"

export async function createPost(formData: FormData) {
  const title = formData.get("title")
  const content = formData.get("content")
  
  await db.posts.create({
    data: { title, content }
  })
  
  revalidatePath("/posts")
}
```

## Uso en Formularios

La forma más común de usar Server Actions es con formularios:

```tsx
import { createPost } from "./actions"

export default function NewPostForm() {
  return (
    <form action={createPost}>
      <input name="title" placeholder="Título" />
      <textarea name="content" placeholder="Contenido" />
      <button type="submit">Crear Post</button>
    </form>
  )
}
```

## Conclusión

Server Actions simplifican enormemente la arquitectura de tu aplicación al eliminar la capa de API tradicional para muchas operaciones comunes.
